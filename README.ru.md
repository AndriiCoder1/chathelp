# Personal AI Chat Assistant (ChatHelp)

Персональный чат-помощник с голосовым вводом и AI-ответами. Проект создан для ежедневного использования в изучении языков и решения задач. Полностью бесплатный, работает через Hugging Face Spaces.

## 🚀 Основные функции

- 🎤 **Голосовой ввод** с автоматической транскрипцией через Whisper
- 💬 **Текстовый чат** с AI-ассистентом (LLaMA 3.1 8B)
- 🌐 **Поиск в интернете** через SerpAPI (погода, курсы, новости)
- 🔊 **Голосовые ответы** через Google TTS
- 🧠 **Умный поиск** с определением типа запроса (цены, товары, события)
- 📝 **Генерация кода** на Python и JavaScript
- 🌍 **Мультиязычность** (русский, английский, немецкий)
- ⏰ **Локальное время и дата** с определением языка
- 💾 **Кэширование ответов** для ускорения
- 🌓 **Темная/светлая тема** интерфейса

## 🛠️ Технологический стек

### Frontend
- **Vanilla JavaScript** (чистый JS без фреймворков)
- **HTML5, CSS3** - адаптивный интерфейс
- **Web Speech API** - распознавание речи в браузере
- **MediaRecorder API** - запись аудио
- **Socket.IO Client** - реальное время

### Backend (Node.js)
- **Express.js** - веб-сервер
- **Socket.IO** - WebSocket соединения
- **SerpAPI** - поисковые запросы
- **Google TTS API** - синтез речи
- **Axios** - HTTP-запросы

### AI и обработка аудио
- **Hugging Face Spaces** - хостинг AI модели
- **LLaMA 3.1 8B** - языковая модель (через Ollama)
- **OpenAI Whisper** - транскрипция аудио (через Hugging Face)
- **gTTS** - синтез речи

## 🏗️ Архитектура проекта

Проект использует распределённую архитектуру:

1. **Клиент (браузер)** - запись голоса, отображение чата
2. **Node.js сервер (Render)** - обработка запросов, поиск, кэширование
3. **Hugging Face Spaces** - AI модель LLaMA 3.1 8B
4. **Hugging Face Inference API** - распознавание речи Whisper

### Схема работы голосового ввода:
Браузер (MediaRecorder) → Node.js → Python (Whisper) → Hugging Face API → текст

### Схема AI-ответов:
Текст → Node.js → Hugging Face Space (LLaMA) → ответ → TTS → браузер

### Поиск в интернете:
Вопрос → определение типа → оптимизация запроса → SerpAPI → AI извлечение ответа


## 📦 Установка и запуск

### Локальная разработка

```bash
# Клонировать репозиторий
git clone https://github.com/AndriiCoder1/chathelp.git
cd chathelp

# Установить зависимости Node.js
npm install

# Установить зависимости Python
pip install -r requirements.txt

# Создать .env файл
cp .env.example .env
# Добавить API ключи: HF_TOKEN, SERPAPI_KEY

# Запустить сервер
npm start

Деплой на Render
Подключить GitHub репозиторий

Добавить переменные окружения:

HF_TOKEN - токен Hugging Face

SERPAPI_KEY - ключ для поиска

Автоматический деплой при пушах

🔑 Необходимые API ключи

HF_TOKEN - получить на huggingface.co/settings/tokens

SERPAPI_KEY - получить на serpapi.com

🧠 Используемые модели

LLaMA 3.1 8B - основная языковая модель (хостится на Hugging Face)

Whisper Large V3 - распознавание речи (через Hugging Face Inference API)
